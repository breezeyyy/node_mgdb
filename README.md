### 开发前的配置要求

- Node 14或以上，vue2
- 使用的第三方node模块请查阅package.json
  
### **安装步骤**

**Clone这个地址： `https://github.com/breezeyyy/node_mgdb.git`**

### 快速入门

- 安装依赖：`npm install`
- 执行`npm start`，
  - [客户端](http://localhost)
  - [管理端](http://localhost/admin)
